# Gemini News App

GoogleニュースのRSSフィードを取得し、生成AI（Gemini API）を用いて自動でタグ付けを行って表示するWebアプリケーションです。

## アプリケーション概要
ユーザーが選択したカテゴリ（テクノロジー、経済、スポーツ、国際）に基づいて最新ニュースを取得します。取得したニュースに対し、バックエンドでAIが記事の内容を分析し、適切なタグを生成して付与します。

## ロジックの説明：カテゴリ変更から画面表示まで
ユーザーがカテゴリを変更してから、実際にニュースとタグが画面に表示されるまでの処理の流れは以下の通りです。

1. **ユーザー操作とリクエスト送信 (Frontend)**
   - ユーザーが画面上のカテゴリボタン（例：「経済」）をクリックします。
   - Vue.jsがクリックイベントを検知し、ローディング表示（`isLoading`）を有効にします。
   - フロントエンドからバックエンド（Flask）のAPIエンドポイント（`/api/news/<category>`）へ非同期リクエストを送信します。

2. **RSSフィードの取得と解析 (Backend)**
   - Flaskサーバーはリクエストを受け取り、指定されたカテゴリに対応するGoogleニュースのRSS URLにアクセスします。
   - 取得したXMLデータを解析し、記事のタイトル、リンク、概要を抽出します。

3. **AIによるタグ生成 (Backend)**
   - 抽出した記事ごとに、Gemini APIへリクエストを送信します。
   - プロンプトとして「記事タイトル」と「記事概要」を渡し、内容を表すタグ（例: "市場, 株価, 輸出"）の生成を依頼します。
   - APIからの応答（文字列）をカンマ区切りでリスト形式に変換します。

4. **レスポンスと画面描画 (Frontend)**
   - Flaskサーバーは、記事情報と生成されたタグをJSON形式にまとめてフロントエンドへ返却します。
   - Vue.jsがデータを受け取り、記事リスト（`articles`）を更新します。
   - DOMが更新され、ニュース一覧とAIが生成したタグが画面に表示されます。

## 動作環境とインストール手順

Webサイトを動作させるためには、以下の手順で環境構築を行ってください。

### 前提条件
- Python 3.10 以上
- Google Cloud ProjectおよびGemini APIキーの取得

### 1. 必要なライブラリのインストール
ターミナル（コマンドプロンプト）を開き、プロジェクトのディレクトリで以下のコマンドを実行します。

```bash
pip install flask flask-cors python-dotenv requests google-genai
```
### 2. 環境変数の設定
プロジェクトのルートディレクトリに `.env` という名前のファイルを作成し、取得したGemini APIキーを以下のように記述して保存します。

```text
GEMINI_API_KEY=ここに取得したAPIキーを記述
```
### 3. バックエンドサーバーの起動
ターミナルで以下のコマンドを実行し、Flaskサーバーを立ち上げます。
```bash
pip python app.py
```
成功すると、ターミナルに以下のようなメッセージが表示されます。
Running on http://127.0.0.1:5000
### 4. アプリケーションの利用
index.html ファイルをWebブラウザ（Chrome, Edgeなど）で直接開いてください。
（VS Code等のLive Server拡張機能を使用しても動作します）
画面が表示され、カテゴリボタンを押すとニュースの取得とAIによる解析が始まります。
